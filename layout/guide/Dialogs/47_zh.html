基本脚本编辑器对话框#

💃访问方式：工具 > 基本脚本 > 基本脚本编辑器菜单项

Basic 是一种简单的脚本语言。与许多 Windows 应用程序（如 Microsoft Word 和 Excel）类似，PADS 应用程序包含 Basic 功能，允许用户使用标准脚本语言自定义应用程序。

您可以使用基本脚本编辑器在 PADS 应用程序中创建、编辑、运行和调试 Basic 脚本。
